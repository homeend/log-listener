## Positional toggles (not individually overridable)

- `1`–`9` — toggle the corresponding group on/off.
- `!@#$%^&*(` (Shift+1..9) — toggle the corresponding renderer on/off.

## macOS note

`Ctrl`+Arrow is captured by macOS Mission Control / Spaces, so the macOS default advertises `shift+down` / `shift+up` / `shift+left` / `shift+right` for fast scrolling (Ctrl+Arrow stays bound as a fallback). **Shift+Arrow forwarding to a TUI has not been verified on every macOS terminal — if it does nothing, PgUp/PgDn still page.**

## Overriding keys in your config

Add a `keybindings:` block to your `log-listener.yml`. Precedence per action: current-OS section → `default` section → built-in default. A listed action's key list fully replaces the lower layer.

```yaml
keybindings:
  default:            # applies on every OS
    search: ["/"]
  darwin:
    fast_down: ["shift+down"]
  linux:
    fast_down: ["ctrl+down"]
  windows:
    fast_down: ["ctrl+down"]
```

Valid action names: `quit`, `toggle_files`, `toggle_groups`, `toggle_renderers`, `close_overlay`, `search`, `next_match`, `prev_match`, `filter`, `toggle_group_col`, `toggle_file_col`, `clear`, `collapse_all`, `scroll_up`, `scroll_down`, `page_up`, `page_down`, `fast_up`, `fast_down`, `top`, `bottom`, `scroll_left`, `scroll_right`, `fast_left`, `fast_right`, `reset_horiz`, `save_viewport`, `save_scrollback`, `next_block`, `prev_block`, `next_marked_block`, `prev_marked_block`, `toggle_exception_marks`, `copy_reference`, `copy_text`, `visual_select`, `toggle_filename_trunc`, `toggle_word_wrap`, `dump_debug`, `help`.
